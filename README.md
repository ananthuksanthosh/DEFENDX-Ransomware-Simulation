# DEFENDX – Dummy Ransomware Simulation

## Overview

This project was developed as part of the IPSR DEFENDX Cyber Security Challenge. The objective of the project is to create a safe and educational ransomware-style simulation using Python and Tkinter inside a controlled virtual machine environment.

The program simulates a ransomware lock screen interface with password-based unlocking functionality for cybersecurity learning purposes only.

---

## Features

* Fullscreen ransomware-style interface
* Password-protected unlock system
* Warning and information messages
* Invalid attempt counter
* Safe educational simulation
* No real file encryption or system modification

---

## Technologies Used

* Python 3
* Tkinter GUI Library
* Kali Linux 2025.2
* Windows 10
* Oracle VM VirtualBox

---

## Disclaimer

This project is strictly for educational and cybersecurity training purposes only.

The program does NOT:

* Encrypt files
* Delete data
* Modify system settings
* Perform malicious activities

The simulation was tested only inside isolated virtual machine environments.
